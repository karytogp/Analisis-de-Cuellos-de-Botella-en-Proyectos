# ⏱️ Análisis de Cuellos de Botella en Proyectos | Project Bottleneck Analysis

## 📌 Descripción
Este proyecto presenta un análisis de tiempos en las diferentes fases de ejecución de proyectos, con el objetivo de identificar cuellos de botella y oportunidades de mejora en los procesos.

El caso está basado en una experiencia real en gestión de proyectos. Por motivos de confidencialidad, solo se comparten visualizaciones y resultados del análisis.

## 🎯 Objetivo
Evaluar la duración de cada fase del proyecto y detectar en qué etapas se generan retrasos que impactan la ejecución.

## 🛠 Herramientas utilizadas
- Power BI
- Excel

## 📊 Enfoque del análisis

Se analizaron las siguientes fases:

- Pre-aprobación  
- Aprobación  
- Logística (compras)  
- Ejecución  

Se calcularon indicadores como:

- Tiempo promedio por fase  
- Comparación entre proyectos  
- Evolución de tiempos  
- Identificación de desviaciones  

## 📈 Insights clave

- La fase logística presentó los mayores tiempos promedio (~78 días)
- Se evidenció un incremento en los tiempos de aprobación (de ~4 a ~17 días)
- Alta variabilidad en algunas fases, afectando la planeación
- Los retrasos en etapas iniciales impactan directamente la ejecución total
